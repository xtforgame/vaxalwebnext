# Vaxal (思序網路) 2026 網站設計提案 (Web Design Proposal)

本提案旨在為 Vaxal 打造一個「詳盡、現代化且具信賴感」的企業官網，透過 **Product-Led Service (產品化服務)** 的敘事策略，將深度的技術顧問服務與強大的 AI 產品線完美結合。

---

## 1. 核心策略與定位 (Core Strategy)

### 1.1 品牌定位：The Pragmatic Growth Partner
Vaxal 不只是一間接案公司，而是**「結合大企業經驗與新創彈性」**的成長夥伴，更是**「反對 AI 泡沫的務實主義者」**。

*   **Anti-Bubble (反泡沫)**：不賣未來的空泛大餅。我們承諾每一階段都有「可量化」的實質改善。
*   **Anti-Lock-in (反綁架)**：我們提供完整資料導出（含 AI 對話紀錄）。客戶留下是因為**信賴**，而非被迫依賴。
*   **Dogfooding (自身實踐)**：我們自家的 3 個 SaaS 產品，皆是使用 Vibe Coding 與 AI 原生工作流開發而成——這就是最強大的 Proof of Work。

### 1.2 核心目標
*   **Trust Building (信任建立)**：透過詳盡的案例與工具演示，建立專業形象。
*   **Lead Generation (潛在客戶轉化)**：透過 3-Step 試點模型吸引各階段企業。
*   **Talent Attraction (人才招募)**：展示「Vibe Coding」與高效遠端協作的企業文化。

### 2.4 案例專頁 (Case Study)

#### Case 1: Studio Doe — 從電商重建到企業數據中心的四年旅程

**Title**: Decoding the Infrastructure behind a Leading Fashion Brand.

**背景**：
Studio Doe 最初找上思序網路，目的僅僅是重寫一套電商系統。然而，思序網路並沒有在相同預算下選擇走「套用成熟電商框架、快速交付」的常見路線。

**關鍵決策 — 以數據為中心的長期架構**：
身兼創業家、顧問與開發商三重角色，思序網路在不影響眼前上線目標的前提下，從第一天起就以「數據中心」為核心，打造了一套高度模組化的底層架構。電商平台本身，只是這套系統中數個模組組合後的產物之一。

這就是思序網路最大的差異化——不做一次性的交付，而是在每一次合作中埋下長期成長的種子，與客戶建立真正的戰略夥伴關係。

**四年演進 — 模組化的力量**：
隨著 Studio Doe 的團隊規模成長至十倍以上，這套系統不僅沒有被淘汰或推翻重來，反而持續穩健地擴充：
*   數據分析模組：營運決策的數據支撐。
*   線下門市整合模組：線上線下庫存、會員、訂單的無縫串聯。
*   代理分潤模組：因應多元銷售渠道的拆帳需求。
*   企業私有 LLM AI 模組 (2025)：將 AI 能力原生整合進既有的營運流程。

從一套電商系統，進化為承載整間企業核心運作的數據中心建設。

**關鍵轉折 — AI 時代的降臨**：
四年前，沒有人能預測 LLM AI 會橫空出世、改寫整個產業的遊戲規則。思序網路自然不是預言家，但我們深知：唯有扎實的基礎架構，才能讓企業在任何浪潮來臨時做好最佳準備。

當 AI 浪潮真正到來，Studio Doe 成為了台灣少數同時具備三項條件的品牌：**自研電商系統、掌握完整第一方行銷數據、以及足夠扎實的可擴充架構**。這讓我們得以在極短時間內，讓 AI 直接產出各類營運分析與報告，迅速成為同業中的領先者。

當其他品牌還受限於第三方電商平台、無法自由開發新功能時，Studio Doe 已經透過 Vibe Coding 大幅甩開差距——而這個優勢只會持續擴大。

> *「沒有人知道 AI 會這樣改寫人類的歷史，但有人能夠在這之前就做好萬全準備。」*

**與 AI 生態的原生相容**：
得益於初始的架構遠見，這套系統與 AI 生態高度相容。開發的每一項功能模組都能簡單地提供對應的 MCP (Model Context Protocol) 服務，原生支援 AI 人機協作，讓企業時刻與 AI 前沿技術無縫接軌，而無需大規模的系統改造。

**核心價值**：
*   證明思序網路「顧問式開發」的長期價值——同樣的預算，截然不同的戰略眼光。
*   證明模組化架構在真實商業環境中的可擴充性——團隊十倍成長，零次砍掉重練。
*   證明傳統系統也能與 AI 無痛接軌——前提是一開始就做對架構決策。
*   證明「自研 + 數據自主」在 AI 時代的決定性優勢——當別人還在等平台開放 API，你已經在用 AI 跑報告了。

**可用附圖**：線上與線下 APP 畫面截圖（展示無縫整合）、模組化架構示意圖。

**客戶推薦語 (Testimonial，待取得)**：
> *建議方向：「我們當初只是要做一個電商網站，但思序幫我們做的是一套能一路跟著我們長大的系統。四年過去，團隊成長了十倍，系統從來沒有需要砍掉重練——甚至今年直接接上了 AI，這是當初完全沒想到的。」*

#### Case 2: Vaxal (Internal) — 自我進化的生態系

**核心敘事：共生共長的閉環**

**第一層：Vaxal Base — 共用的模組化基底系統**
*   Vaxal Base 是思序自研的模組化基底系統，同時服務自身業務與合作夥伴。
*   「與客戶一同成長」不只是口號——因為 Vaxal 自己的業務需求會不斷驅動 Vaxal Base 的維護與升級，客戶的許多模組升級和新功能開發甚至是免費的。
*   改進成果主動免費推薦給合作夥伴，協助他們擴展新的可能性。
*   客戶不用擔心系統交付後無人維護，也不用擔心新模組的開發斷炊。

**第二層：從 Vaxal Base 衍生出獨立產品**
*   Ryko、BrevFlow、SoloistBoard 三個產品，本質上是從 Vaxal Base 的自用模組衍生而來。
*   它們可以獨立作為服務推給新客戶。
*   對於已使用 Vaxal Base 的客戶，則能獲得**原生整合支援**，無縫套用。

**第三層：自我進化的閉環**
*   這三個產品正是 Vaxal 持續開發 Vaxal Base 的重要工具——生態系自己讓自己進化。
*   進化成果共享給夥伴，形成正向循環。

**敘事方向**：不是「我們也用自己的產品」的表面 dogfooding，而是一個自我強化的生態系——Vaxal Base 孵化出產品，產品反哺 Vaxal Base，成果與夥伴共享。

### 2.5 關於我們 (About) — 「我們是誰、為什麼這樣做」

**定位**：精煉版公司介紹，讓訪客快速建立對團隊的信任。與 Vaxal Base 頁互補——About 建立「人」的信任，Vaxal Base 建立「方法論」的信任。

**頁面結構**：

| 區塊 | 內容 |
|------|------|
| **Hero** | 保留現有標題結構，微調文案呼應「堅實夥伴」定位 |
| **Origin（新增）** | 創辦人口吻的簡短起源敘事：為什麼我們在 2017 年選擇自建一套系統，而不是用現成框架接案 |
| **Milestones（新增）** | 時間軸精華版：2017 首版完成 → 同年七月八維智能成為第一個夥伴 → 9 年間 32 家合作、153 個模組、橫跨 7+ 產業 → 6 家深度互惠夥伴（其中 3 家年營收/資本額數億） |
| **Core Values** | 保留三個核心價值，重新措辭讓它們對應到實際故事而非抽象口號 |
| **Team DNA + Founders** | 保留現有結構（團隊基因介紹 + 4 位創辦人卡片） |
| **Bridge CTA（新增）** | 「了解驅動這一切的系統 →」導向 Vaxal Base 頁面 |

### 2.6 Vaxal Base 頁 — 「一個自我進化的生態系」

**定位**：創辦人自述，Editorial 風格（同 Studio Doe 的敘事調性）。不是產品介紹頁，而是傳達「我們為什麼這樣解決問題」的方法論故事。

**頁面結構**：

| 區塊 | 內容 |
|------|------|
| **Hero** | 標題 + 一句話定位 Vaxal Base。不是推銷，是「我們為什麼這樣解決問題」 |
| **Ch1：起點** | 2017，為什麼我們決定自建而不用現成方案。不是因為喜歡造輪子，而是我們發現每一次交付後系統就死了——沒人維護、無法擴展。我們想解決這個問題 |
| **Ch2：共生** | Vaxal Base 同時服務我們自己和夥伴。我們的業務需求驅動升級，成果免費共享。客戶不用擔心斷炊，因為我們自己每天都在用 |
| **Figure：產業廣度** | 視覺呈現 7+ 產業（牙醫、電商、教育、虛擬貨幣交易、自動化監控、線上客服、AI SaaS）— 同一套基底，153 個模組的排列組合 |
| **Ch3：衍生** | 從自用模組長出 Ryko、BrevFlow、SoloistBoard（點到名字和角色，不展開功能細節）。對既有夥伴是原生整合，對新客戶是獨立服務 |
| **Ch4：閉環** | 這三個產品反過來成為開發 Vaxal Base 的工具。生態系自己讓自己進化，進化成果共享給夥伴 |
| **數字區塊** | 2017 年至今 · 32 家合作夥伴 · 153 個模組 · 6 家深度互惠 · 3 家數億規模企業 |
| **CTA** | 呼應 About 頁調性，邀約合作 |

**關鍵數據**：
*   首版完成：2017 年
*   第一個合作夥伴：八維智能（2017 年 7 月）
*   至今合作：32 家公司，6 家長期深度互惠，其中 3 家年營收/資本額數億
*   模組數量：153 種
*   涵蓋產業：牙醫系統、線上電商、教育平台、虛擬貨幣交易、自動化監控、線上客服、AI SaaS

**兩頁敘事邏輯**：About 說「我們是這樣的團隊」→ Vaxal Base 說「這是我們為什麼能這樣做」。About 建立人的信任，Vaxal Base 建立方法論的信任。
